Microsoft Face API


Microsoft Face API - наиболее известный сервис по распознаванию лиц предоставляет компания Microsoft, который называется Face API. Это облачный сервис, который предоставляет большинство продвинутых алгоритмов по распознаванию лиц людей. Face API умеет распознавать на одном изображении до 64 человеческих лиц с высокой точностью их расположения. Изображение может быть отправлено сервису в виде файла или адреса картинки на другом сервере. Результатом являются координаты расположения найденного лица, а также пол, возраст, эмоции, наличие/отсутствие волос или очков.

    Прмеры сфер использования технологии распознавания лиц:

  • Безопасность
  • Анализ и менеджмент содержимого изображений
  • Мобильные приложения
  • Робототехника
